Command-Line Befehle

Befehl Englische Bezeichnung Funktion
ls list listet den Inhalt eines Ordners auf
pwd print working directory zeigt den Ordner an, in welchem man sich gerade befindet
cd change directory in angegebenen Ordner wechseln
cp copy kopiert die Dateien/den Ordner
mv move verschiebt Dateien und Ordner
rm remove löscht Dateien/Ordner
mkdir make directory erstellt einen neuen Ordner
rmdir remove directory löscht einen Ordner
open / öffnet die angegebene Datei
kill/killall / beendet einen angegebenen Prozess
whoami / zeigt den aktuellen Benutzer an
su switch user wechselt den aktuellen Benutzer
sudo substitute user do führt den Befehl als Superuser/Admin (root) aus
pbcopy pasteboard copy übergibt Inhalt an die Zwischenablage von OS X
pbpaste pasteboard past gibt Inhalt der OS X Zwischenablage aus
say / Sprachausgabe eines eingegebenen Textes
date / zeigt das aktuelle Datum und Uhrzeit an
screencapture / erstellt ein Bildschirmfoto
find / Suche nach Dateien
mdfind / Spotlight-Suche
ps process status gibt die aktuell aktiven Prozesse aus
top / listet eine detaillierte Prozessliste auf
history / listet die zuletzt benutzten Befehle auf
clear / verschiebt die Promt so, dass im aktuellen Terminal-Fenster alles leer ist
man man-page ruft die Man-Seite zum angegebenen Befehl auf
which / welcher Befehl wird ausgeführt
whatis / Kurzinfo zum angegebenen Befehl
foundation watch / automatische Aktualisierung (durch speichern) des Webbrowsers, wenn Änderungen im Code vorgenommen werden
foundation new / anlegen eines neuen Projektes mit Foundation
