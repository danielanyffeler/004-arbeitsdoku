Statische Webseite

Pro

  • benötigt keine Datenbankverbindung – Elemente direkt in HTML-File integriert
  • Man benötigt nur HTML und CSS Kenntnisse
  • Sehr sicher vor Hackerangriffen
  • Statische Webseiten sind in der Regel schneller aufgebaut, als dynamische, da keine Verbindung zur Datenbank hergestellt werden muss
  • keine grossen Anforderungen an Webserver – kostengünstig

Con

  • jede Änderung muss direkt in dem entsprechenden HTML-File vorgenommen werden
  • Bei vielen Seiten wird das Projekt sehr schnell unübersichtlich
  • Designänderungen vorzunehmen ist umständlicher als bei einer dynamischen Webseite
  • je grösser das Projekt, desto mehr Aufwand muss betrieben werden
