Foundation – XY Grid

Basics

Die Struktur wird hauptsächlich mit den Klassen .grid-x .grid-y und .cell definiert. .grid-x und .grid-y sind dazu da, die Richtung, wie die Elemente angereiht werden sollen. .cell gibt dem Element eine Höhe und Breite. .grid-x und .grid-y kommen dann zum Einsatz, wenn man eine Neue Zeile/Abschnitt definiert, in welchem die Elemente anders angereiht werden. Innerhalb arbeitet man dann mit .cell's und füllt diese mit dem entsprechenden Inhalt.

Klasse Auswirkung
.grid-x bewirkt, dass das Element horizontal angeordnet wird
.grid-y bewirkt, dass das Element vertikal angeordnet wird
.cell Gibt dem Element ein Volumen (Höhe und Breite)
.grid-container nimmt das grösstmögliche Volumen ein
wird zentriert

Gutters

Ohne Gutters werden die Elemente ohne Abstand aneinander gereiht. Um einen Abstand durch margin/padding zu erzeugen, verwendet man die Klassen .grid-margin-x / .grid-padding-x.

Grid Container

Das Grid passt sich auf die ganze Breite des verfügbaren Platzes an (.grid-container). Der Container wird zentriert und hat die gleiche max-width, wie $grid-container.

Fluid

Bewirkt, dass das Element auf die gesamte Breite des Verfügbaren Platzes verteilt wird.

Full

Hiermit wird das Padding überschrieben. Kommt primär bei grid-marin-x zum Einsatz; bei grid-padding-x hat es die gleiche Wirkung, wie fluid.

Auto Sizing

Mit der Klasse .auto kann man den Noch verfügbaren Platz ausfüllen. Wenn zwei Elemente mit dieser Klasse versehen werden, wird der Platz gleichmässig auf die Elemente aufgeteilt.

Horizontal und vertikal zentrieren

div mit den Klassen .grid-x und .grid-y versehen diesem div die Werte height: 100vh und justify-content: center geben.

Sticky

Mit sticky kann man ein Element fix positionieren, so dass es immer an der selben Stelle bleibt bzw. mitscrollt.
<div class="columns small-6 right" data-sticky-container>
<div class="sticky" data-sticky data-margin-top="0">
<!-- Hier kommt das Element rein -->
</div>
</div>

Accordion Menu
