Pro

MarkDown Definition

MarkDown ist eine vereinfachte Art Text auf einer Webseite zu formatieren und wird vor allem von Bloggern und IT-Einsteiger benutzt. Anstelle von eckigen Klammern, Befehlen oder sonstigen Kommandos wie bei einem HTML-File, werden hier einfache Zeichen zur Formatierung verwendet. Z.B. Sternchen, Underscores, Hashtags etc. Dies führt dazu, dass der Text auch im Rohformat noch angenehm zu lesen ist. MarkDown wird auch als eine Art Anti-Computersprache bezeichnet, da sie in einer gewissen Weise für einige Computersprachen überflüssig macht. Fazit: Sie wurde vor allem für Nicht-Programmierer entwickelt.

Beispiele

  • *kursiv*
  • **fett**
  • #H1
  • ##H2
  • ###H3
